School Management System (Maluti Primary School)
Overview
This is a web-based School Management System developed using PHP, MySQL, HTML, CSS, and JavaScript. The system is designed to help manage daily school operations such as student records, fees, attendance, and grades.
Features
•	Student Management (Add, edit, delete students)
•	Class Management
•	Subject Management
•	Teacher Management
•	Attendance Tracking
•	Grades and Report Cards
•	Fee Management
•	Authentication System (Login/Register)
•	Notifications
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP / Apache
Installation & Setup
1.	Extract or clone the project into the htdocs folder (XAMPP).
2.	Start Apache and MySQL in XAMPP.
3.	Create a database in phpMyAdmin ( maluti_primary_school).
4.	Import the SQL file from the database folder.
5.	Configure db.php with your database credentials.
6.	Run the system in browser using localhost.
7.	Use register.php as an index is unavailable
Project Structure
Main files include: add_student.php, manage_students.php, dashboard.php, login.php, register.php, db.php, session.php, and database SQL file.
